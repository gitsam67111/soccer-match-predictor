# Soccer Match Predictor

A machine learning project that predicts English Premier League match outcomes using historical data and Elo ratings.

## Features

- Random Forest Classifier
- Elo rating system
- Previous 5-match form
- Home advantage
- Match outcome prediction

## Accuracy

Current accuracy: 44%

## Future Improvements
